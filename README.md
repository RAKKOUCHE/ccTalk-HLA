# HLA-ccTalk
  ccTalk Bus Saleae High Level Analyser

## Screens shot
  ![Screenshot](Screens.pdf)

## Feature
  * Decode Device to master and master to device
  * Add one instance for each device 
  ### Limitation
    This version does not decode:
      * MDCES commands
      * Broadcast commands
      * Encrypted commands
